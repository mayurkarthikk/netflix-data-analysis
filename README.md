# Netflix Movies and TV Shows Data Analysis

This project analyzes the Netflix movies and TV shows dataset to derive actionable insights for content strategy and business growth.

## Dataset

The dataset contains listings of all movies and TV shows available on Netflix, including details such as:
- Show ID
- Type (Movie/TV Show)
- Title
- Director
- Cast
- Country
- Date added
- Release year
- Rating
- Duration
- Genre (listed_in)
- Description

Source: [Netflix Movies and TV Shows dataset](https://www.kaggle.com/datasets/shivamb/netflix-shows) (Kaggle)

## Project Structure

- `netflix.csv` - The raw dataset
- `netflix_data_analysis.ipynb` - Jupyter notebook containing the complete analysis

## Analysis Overview

The notebook covers:
1. **Data Exploration** - Understanding the dataset shape, data types, missing values
2. **Data Cleaning** - Handling missing values, data type conversions
3. **Exploratory Data Analysis (EDA)**
   - Univariate analysis: distribution of content types, ratings, release years, etc.
   - Bivariate analysis: relationships between variables
   - Geographic analysis: content availability by country
   - Temporal analysis: trends over time
4. **Insights & Recommendations**
   - Content type trends (Movies vs TV Shows)
   - Popular genres and ratings
   - Geographic content distribution
   - Temporal patterns in content addition
   - Actor/director analysis
5. **Business Recommendations**
   - Content strategy suggestions
   - Market expansion opportunities
   - Recommendation improvements
